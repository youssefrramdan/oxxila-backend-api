# my-express-app

Simple Express starter project with a scalable folder structure.

## Structure

- `src/config` - configuration files and environment settings
- `src/controllers` - request/response handlers
- `src/models` - data models and schemas
- `src/routes` - route definitions
- `src/middlewares` - custom middleware
- `src/services` - business and integration services
- `src/utils` - helpers and shared utilities
- `src/app.js` - app composition
- `src/server.js` - server bootstrap

## Run

```bash
node src/server.js
```
